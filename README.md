# inventoryManager

A Vue 3 application for managing inventory products with full CRUD operations.

## Technologies Used

- Vue 3 (Composition API)
- Vite
- PrimeVue (UI components)
- Pinia (state management)
- TanStack Vue Query (data fetching and caching)
- Axios (HTTP client)
- Zod (schema validation)
- TypeScript
- Tailwind CSS

## Installation

```sh
npm install
```

## Running the Application

```sh
npm run dev
```

The application will be available at `http://localhost:5173`.

Ensure the backend API is running on `http://localhost:8080` with CORS enabled for the frontend origin.
